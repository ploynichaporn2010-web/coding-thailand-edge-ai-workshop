## 2. Class Definition

ระบุ class ที่จะใช้ + เกณฑ์ที่ทำให้แต่ละ class แตกต่าง:

| # | ชื่อ Class | คำอธิบาย (3-4 ประโยค) | สิ่งที่ทำให้ต่างจาก class อื่น |
|---|---|---|---|
| 1 | metaal | เสียงกระทบโลหะ/กระป๋ออลูมเนียม | เสียงแหลม กังวาน | 
| 2 | plastic | เสียงกระทบขวด/ภาชนะพลาสติก | เสียบทบ ไม่กัววาน | 
| 3 | noise | เสียงรบกวนทั่วไป/background | เสียงพูด ลม พื้นหลัง | 

**ตัวอย่าง (Track A, Fall Detector):**

| # | ชื่อ Class | คำอธิบาย | สิ่งที่ทำให้ต่าง |
|---|---|---|---|
| 1 | metal | เสียงกระทบกระป๋องโลหะ | เสียงแหลม กังวาน ความถี่สูง |
| 2 | plastic | เสียงกระทบขวดพลาสติก | เสียงทึบ ความถี่ต่ำ สน |
| 3 | noise | เสียงรบกวน background | ไม่มี pattern ชัดเจน |

## 3. Edge Cases — สิ่งที่อาจสับสน

ระบุสถานการณ์ที่อาจทำให้โมเดลสับสน:

```
1. กระป๋องโลหะที่บุบหรือห่อด้วยพลาสติก -เสียงกระทบเพี้ยน อาจถูกจัดเป็น
   ตัวอย่าง: "นั่งลงเร็วๆ" อาจถูกจัดเป็น "ล้ม"

2. ขวดพลาสติกแข็งหนา (เช่น PP/HDPE) -เสียงคล้ายโลหะ อาจถูกจัดเป็น metal

3. ทิ้งขยะพรองกันพรองหลายชี้น / เสียบดังรอบข้าง -เสียงซอนกัน โมเดลสับสนระหว่าง class
``` 

**วางแผนแก้:**ยบ
```
[1 ถ้า confidence < 80% -"ไม่ trigger servo รอฟงใหม่ 2 เก็บเสียงในหลายสภาพแวดล้อง (เงียบ/มีเสียงรบกวน) 3 เพิ่ม noise class ให้โมเดลรู้จักเสียงที่ไม่ใช่ขยะ]
```

## 4. Data Collection Plan
| Class | จำนวน samples | ระยะเวลา/sample | สภาพแวดล้อมที่จะเก็บ |
|---|---|---|---|
| metal | 50 samples | 1-2 วินาที | ห้องเรียน/โรงอาหาร |
| plastic | 50 samples | 1-2 วินาที  | ห้องเรียน/โรงอาหาร |
| noise | 50 samples | 1-2 วินาที  | มีเสียงพูด/เสียนรบกวนรอบข้าง |

**Total samples ทั้งหมด:** 150 samples
**คาดว่าใช้เวลาเก็บ:** 30-45 นาที

⚠️ **เป้าหมายขั้นต่ำ:**
- Motion/Audio: 50 samples × class
- Vision: 100 images × class
- Environment: 200 samples × class

## 5. Diversity Plan — กันอคติ (Bias)

ตอบ 3 คำถาม:1 ห้องเรียน (เงียบ) 2 โรงอาหาร (มีเสียงรบกวนปานกลาง) 3 ทางเดิน (มีเสียงคนพูด/เดิน)

### a) มีหลาย "environment" ไหม?
```
[1 ห้องเรียน (เงียบ) 2 โรงอาหาร (มีเสียงรบกวนปานกลาง) 3 ทางเดิน (มีเสียงคนพูด/เดิน)]
```

### b) มีหลาย "subject" ไหม? (กรณี Vision/Audio/Motion)
```

[กระป๋องหลายขนาด (เล็ก/กลาง/ใหญ่),พลาสติกหลายประเภท (ขวด/กล่อง/ถุง),คนทิ้งต่างกัน (โยน/วาง/ทิ้งแรง)]
```

### c) มีหลาย "variation" ไหม?
```
[มุมกระทบ: ตรง/เฉียง,ความเร็ว: ช้า/เร็ว/แรง,ระดับเสียง: ใกล้ไมล์,เวลา:เช้า/กลางวัน (เสียง background ต่างกัน]
```

```
